各組組長透過 Github classroom 連結, 進入後依照組序分別建立 g1, g2, g3 .... 等組別

s5 伺服器

https://s5.eng.nfu.edu.tw:9500 

每位學員可以在 s5 伺服器中取得一個登入帳號，各自將分組倉儲 git clone 至帳號下，採個人分支參與協同設計後，再利用 Pull Requests 與各分組的 main 分支合併.

s5 伺服器中可以靜態伺服倉儲的各版本，也能如 codespaces 讓使用者直接在瀏覽器維護各分支的網站內容。
