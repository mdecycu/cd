但此時 scrum-1 分支尚未存在, 因此利用 git checkout -b 加上分支名稱, 可以同時新增分支且直接切入

Y:\tmp\cd>

這個階段, scrum-1 必須先接受組長送出的倉儲共同維護邀請 (invitation), 同意後才能用 scrum-1 的 SSH session 設定取下協同倉儲

接下來準備將此一階段的版本推向 scrum-1 分支, 其中包括 images 目錄中有 github 帳號下的邀請畫面, 還有這個 scrum-1.txt 檔案
